## Submarines (SSN/SSBN/SSGN) - 66 Ships

Submarine locations are classified and not available on uscarriers.net. The submarine tracker uses hardcoded operational status categories:
- **Deployed** - Currently at sea on patrol/mission
- **Port** - In homeport for maintenance/crew rest
- **Maintenance** - Extended maintenance period
- **Training** - Conducting training operations

### Los Angeles Class (SSN-688) - 26 Active
Hull numbers: SSN688 - SSN773 (with gaps for decommissioned boats)

### Virginia Class (SSN-774) - 23 Active
Hull numbers: SSN774 - SSN800 (with gaps for boats under construction)

### Ohio Class SSBN - 14 Active
Hull numbers: SSBN726 - SSBN743

### Ohio Class SSGN - 4 Active

## URL Pattern Reference

```
Carriers:     http://uscarriers.net/cvn{XX}history.htm
Amphibs LHD:  http://uscarriers.net/lhd{X}history.htm
Amphibs LHA:  http://uscarriers.net/lha{X}history.htm
Destroyers:   http://uscarriers.net/ddg{XX}history.htm
              http://uscarriers.net/ddg{XXX}history.htm (100+)
              http://uscarriers.net/ddg{XXXX}history.htm (Zumwalt)
```

## Homeport Legend

- **ATLANTIC** - East Coast (Norfolk, Mayport)
- **PACIFIC** - West Coast (San Diego, Bremerton, Pearl Harbor)
- **WESTPAC** - Forward Deployed (Yokosuka, Sasebo, Guam)
- **EUCOM** - European Command (Rota, Spain)
